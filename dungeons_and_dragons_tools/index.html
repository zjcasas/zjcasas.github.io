About

Welcome to D&D Tools! D&D Tools provides players with tools and information to assist them in creating new characters, creating encounters, and leveling up their existing characters in Dungeons & Dragons 5th Edition. Stay tuned for future updates. Things I'd like to add include pages for equipment, backgrounds, general rules for playing, monsters, character name generation, random character generation, character management, and potentially more.
